# Arco Design 审查清单

## 审查流程

```
1. 收集信息 → 2. 分类检查 → 3. 问题定级 → 4. 输出报告
```

---

## 一、颜色系统检查

### 主色使用

| 检查项 | 状态 | 说明 |
|--------|------|------|
| 主色是否为 `#165DFF`？ | ☐ | Primary-6 |
| Hover 状态是否为 Primary-5？ | ☐ | `#4080FF` |
| 点击状态是否为 Primary-7？ | ☐ | `#0E42D2` |
| 禁用状态是否为 Primary-3？ | ☐ | `#94BFFF` |
| 是否使用了非 Arco 标准颜色？ | ☐ | 如 Ant Design 的 #1890ff |

### 中性色使用

| 检查项 | 标准色值 | 状态 |
|--------|----------|------|
| 主要文字 | Grey-10 `#1D2129` | ☐ |
| 次要文字 | Grey-8 `#4E5969` | ☐ |
| 占位/禁用文字 | Grey-5 `#AEB5BE` | ☐ |
| 边框 | Grey-3 `#E5E6EB` | ☐ |
| 页面背景 | Grey-1 `#F7F8FA` | ☐ |

### 功能色使用

| 检查项 | 标准色值 | 状态 |
|--------|----------|------|
| 成功状态 | `#00B42A` | ☐ |
| 警告状态 | `#FF7D00` | ☐ |
| 错误状态 | `#F53F3F` | ☐ |
| 链接颜色 | 主色 `#165DFF` | ☐ |

### 可访问性

| 检查项 | 要求 | 状态 |
|--------|------|------|
| 文字与背景对比度 ≥ 4.5:1 | WCAG AA | ☐ |
| 大文字对比度 ≥ 3:1 | WCAG AA | ☐ |
| 禁用状态是否仍可辨识 | 不完全透明 | ☐ |

## 二、字体排版检查

### 字号规范

| 检查项 | 标准 | 状态 |
|--------|------|------|
| 正文是否为 14px？ | 主字号 | ☐ |
| 最小字号是否 ≥ 12px？ | 可识别性 | ☐ |
| 字号层级是否控制在 3-5 种？ | 克制层级 | ☐ |
| 层级差距是否为 2-4px？ | 韵律感 | ☐ |

### 字重使用

| 检查项 | 状态 | 说明 |
|--------|------|------|
| 正文使用 Regular (400)？ | ☐ | 非 Light 或 Bold |
| 标题使用 Medium/Semibold (500/600)？ | ☐ | 非单纯增大字号 |
| 强调使用字重而非字号？ | ☐ | 优先级原则 |

### 行高

| 检查项 | 标准 | 状态 |
|--------|------|------|
| 默认行高为 1.4？ | 14px → 20px | ☐ |
| 中文长文本行高 ≥ 1.5？ | 密度适配 | ☐ |
| 标题行高较小？ | 32-44px | ☐ |

### 字体栈

| 检查项 | 状态 | 说明 |
|--------|------|------|
| 是否包含西文字体？ | ☐ | Nunito, Helvetica Neue |
| 是否包含中文回退？ | ☐ | PingFang SC, Microsoft YaHei |
| 是否包含通用回退？ | ☐ | Arial, sans-serif |

## 三、间距布局检查

### 间距规范

| 检查项 | 状态 | 说明 |
|--------|------|------|
| 间距是否为 4px 的倍数？ | ☐ | 4, 8, 16, 24, 32... |
| 是否使用 Space 组件？ | ☐ | 非 margin 硬编码 |
| 模块间距是否 > 元素间距？ | ☐ | 亲密性原则 |
| 页面边距是否为 24px？ | ☐ | 标准边距 |

### 内边距规范

| 组件 | 标准内边距 | 状态 |
|------|------------|------|
| 按钮 | 6px 16px | ☐ |
| 卡片 | 16px / 24px | ☐ |
| 表格单元格 | 12px 16px | ☐ |
| 输入框 | 4px 12px | ☐ |
| 弹窗 | 24px | ☐ |

### 栅格使用

| 检查项 | 状态 | 说明 |
|--------|------|------|
| 是否使用 24 列栅格？ | ☐ | Row + Col |
| 栅格间距是否统一？ | ☐ | gutter 一致 |
| 是否响应式适配？ | ☐ | xs/sm/md/lg/xl |

### 对齐一致性

| 检查项 | 状态 | 说明 |
|--------|------|------|
| 表单标签对齐方式是否统一？ | ☐ | 全水平或全垂直 |
| 按钮组对齐是否一致？ | ☐ | 左/右/居中 |
| 页面标题对齐是否统一？ | ☐ | 全左或全居中 |

## 七、问题定级标准

### P0 - 严重问题（必须修复）

**定义：** 违反核心价值观，影响可用性

| 问题类型 | 示例 |
|----------|------|
| 可用性 | 用户无法完成核心操作 |
| 可访问性 | 对比度不足，色盲无法区分 |
| 一致性 | 同类功能交互完全不同 |
| 颜色 | 使用了完全错误的主色 |
| 安全 | 危险操作无确认 |

### P1 - 中等问题（建议修复）

**定义：** 违反设计原则，影响体验

| 问题类型 | 示例 |
|----------|------|
| 组件使用 | 按钮层级混乱 |
| 间距 | 间距不符合 4px 基数 |
| 颜色 | 使用了非标准色值 |
| 反馈 | 操作无反馈 |
| 表单 | 验证不完善 |

### P2 - 优化建议（可选）

**定义：** 符合规范但可以更好

| 问题类型 | 示例 |
|----------|------|
| 排版 | 可以优化行高 |
| 间距 | 可以调整模块间距 |
| 交互 | 可以增加动画 |
| 文案 | 可以更友好 |
