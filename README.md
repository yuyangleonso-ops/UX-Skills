# Arco Design Guide Skill

一个专业的 Arco Design 设计系统指导与审查工具，帮助开发者构建符合字节跳动企业级设计规范的中后台界面。

---

## 简介

**Arco Design** 是字节跳动出品的企业级设计系统，核心理念是**"务实的浪漫主义"**——在解决实际问题的同时注入美感与韵律。

这个 Claude Code Skill 提供两大核心能力：

- **🎨 Guide（设计指导）**：为现代简洁的 UI/UX 提供设计原则、具体操作规则和代码示例
- **🔍 Review（设计审查）**：审查现有界面（截图/代码）并输出优先级明确、可操作的修复方案

---

## 功能特性

### 1. 设计指导（Guide）

当创建新界面时，提供：

- **场景分析**：识别页面类型（列表页/表单页/详情页/Dashboard）
- **布局建议**：基于 24 列栅格的布局方案
- **组件选择**：推荐合适的 Arco 组件及组合模式
- **规范指导**：颜色、字体、间距的精确使用规则
- **代码示例**：可直接使用的 React/Vue 代码

### 2. 设计审查（Review）

当评估现有设计时，提供：

- **规范检查**：逐项检查颜色、字体、间距、组件使用
- **问题定级**：按 P0/P1/P2 分级（严重/中等/建议）
- **修复方案**：具体的修改建议和代码示例
- **审查报告**：结构化的审查报告模板

### 3. 完整的规范体系

包含 Arco Design 的核心规范：

- **四大价值观**：清晰、一致、韵律、开放
- **八大设计原则**：及时反馈、贴近现实、系统一致性等
- **颜色系统**：13 种主色、10 级色板、功能色、中性色
- **字体排版**：字号层级、字重系统、行高规范
- **间距布局**：4px 基数、24 列栅格、响应式断点
- **组件指南**：Button、Table、Form、Modal 等核心组件最佳实践
- **页面模式**：列表页、表单页、详情页、Dashboard 完整模板

## 使用方法

### 触发方式

当对话中出现以下内容时，Skill 会自动触发：

```
- "Arco Design" / "字节跳动组件库" / "arco"
- "中后台管理系统" / "企业级产品" / "B端界面"
- "审查 UI" / "检查设计规范"
- React/Vue 管理后台相关的设计需求
```

### 使用示例

#### 场景 1：设计新页面

**用户输入：**
```
帮我设计一个用户管理列表页面，使用 Arco Design 风格
```

**Skill 会提供：**
- 标准列表页布局结构
- 筛选区、表格、分页的组件组合
- 颜色使用规范（Primary-6 #165DFF 等）
- 间距规范（16px、24px 等 4px 倍数）
- 完整的 React 代码示例

#### 场景 2：审查现有设计

**用户输入：**
```
请审查这个页面设计：
- 主按钮颜色：#1890ff
- 表格没有设置 rowKey
- 两个 Primary 按钮并排
- 使用了 13px 和 15px 的字号
```

**Skill 会输出：**
```markdown
## 审查报告

### 严重问题（P0）
1. **主色错误**
   - 当前：#1890ff（Ant Design 颜色）
   - 应为：#165DFF（Arco Primary-6）
   - 修复：修改 CSS 变量或主题配置

### 中等问题（P1）
1. **按钮层级混乱**
   - 问题：同一区域两个 Primary 按钮
   - 规范：每区域最多 1 个 Primary
   - 修复：次要操作改为 Default 或 Text

...
```

#### 场景 3：咨询设计规范

**用户输入：**
```
Arco Design 的字体规范是什么？行高应该用多少？
```

**Skill 会回答：**
- 主字号 14px，最小 12px
- 行高默认 1.4，中文长文本 1.5-1.8
- 用字重（500/600）而非字号来强调
- 提供具体的 CSS 代码

## 设计原则速查

### 四大价值观

| 价值观 | 含义 | 实践 |
|--------|------|------|
| **清晰** | 减少不确定因素 | 操作流程一步到位 |
| **一致** | 样式规范统一 | 降低用户学习成本 |
| **韵律** | 排版有规律美感 | 4px 基数间距系统 |
| **开放** | 灵活可配置 | 支持主题定制 |

### 快速决策表

| 决策项 | 推荐值 |
|--------|--------|
| 主色 | `#165DFF` (Primary-6) |
| Hover | `#4080FF` (Primary-5) |
| Active | `#0E42D2` (Primary-7) |
| 正文 | `#1D2129` (Grey-10, 14px) |
| 次要文字 | `#4E5969` (Grey-8) |
| 边框 | `#E5E6EB` (Grey-3) |
| 页面背景 | `#F7F8FA` (Grey-1) |
| 间距基数 | 4px |
| 页面边距 | 24px |
| 栅格系统 | 24 列 |
